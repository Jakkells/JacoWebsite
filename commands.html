Commands Guide

Welcome to the Commands Guide! Here you will find detailed explanations of the commands you can use in the game.

Shop Commands

Use the following commands to interact with the shop:

  • shop:buy
  • shop:sell
  • shop:action:item:quality:amount

Example: shop:buy:sword:common:1

Item Qualities

The different qualities of items are as follows:

  • common
  • uncommon
  • rare
  • epic
  • legendary
  • mythic

Items Available

The different items you can find in the game include:

  • sword
  • helmet
  • gloves
  • chestplate
  • greaves
  • boots
  • leggings
  • shield
  • robes
  • wand
